Async Reports for OpenERP

In this moment this is only v5 compatible, if you need it for a modern version you can send a pull request :)

How to install

  1. Install dependencies

    $ pip install -r async_reports/requirements.txt

  2. Install oorq module

  3. Put the module in your OpenERP $ADDONS_PATH

  4. Install the module
